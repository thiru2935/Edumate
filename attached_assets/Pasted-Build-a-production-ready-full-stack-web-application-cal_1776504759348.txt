Build a production-ready full-stack web application called **EduMate** — an AI-powered student learning, focus tracking, and peer mentoring platform.

---

TECH STACK:

Frontend:

* Next.js (App Router)
* Tailwind CSS
* Clean component-based architecture

Backend:

* Node.js with Express OR Next.js API routes
* RESTful APIs

Database:

* PostgreSQL

Authentication:

* JWT-based authentication (NO auto email login)
* Password hashing using bcrypt
* Role-based login (Student, Mentor, Teacher)

---

AUTHENTICATION SYSTEM:

Create a complete auth flow:

1. Signup Page:

* Fields:

  * Full Name
  * Email
  * Password
  * Age
  * Role (Dropdown: Student, Mentor, Teacher)
* Validate inputs
* Hash password before storing
* Save in database

2. Login Page:

* Email + Password login
* Generate JWT token
* Redirect user based on role:

  * Student → /dashboard/student
  * Mentor → /dashboard/mentor
  * Teacher → /dashboard/teacher

IMPORTANT:

* Do NOT bind login to a single email
* Do NOT auto-login users
* Must support multiple users
